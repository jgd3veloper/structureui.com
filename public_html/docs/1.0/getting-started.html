Get Started

Build responsive, mobile-first projects on the web with Structure UI a simple but powerful front-end component library. Structure UI is a modern simple but powerful front-end framework making it easier for developers to develop their projects using Structure UI HTML, CSS, and JS. Get Started using Structure UI with our simple starter page.

Starter Template

Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a utf-8 and a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this:

<!-- HTML Template -->
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph</p>

</body>
</html>
<!-- HTML Template End -->

Meta Tags

Structure UI is designed and developed mobile first, a strategy in which we optimize code for mobile devices first and then scale up components as necessary using CSS media queries. To ensure proper rendering and touch zooming for all devices, add the responsive viewport meta tag to your <head> section.

<!-- Meta Tags -->
<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
<meta charset="UTF-8">
<meta name="author" content=" ">
<meta name="description" content=" ">
<meta name="keywords" content=" ">
<!-- Meta Tags End -->
